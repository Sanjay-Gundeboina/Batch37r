    DEFINATION LIST

    FRONTEND
    The frontend is the part of a website or application that users directly interact with. It includes everything that users see on their screens, such as layouts, buttons, images, text, and navigation menus. It is built using technologies like HTML, CSS, and JavaScript to create the structure, style, and interactivity of a webpage or app.
    BACKEND
    The backend is the part of a website or application that works behind the scenes and is not visible to users. It handles the server, database, and application logic. The backend processes user requests, performs operations like data storage and retrieval, and sends the appropriate response back to the frontend. It is built using languages like Python, Java, PHP, Node.js, and uses databases like MySQL, MongoDB, or PostgreSQL.
